Features

Everything you need for batch renames.

From quick prefix swaps to regex-driven cleanups across entire directories.

⌘

File Browser

Navigate the full filesystem. Directories first, then files — alphabetically sorted. Tap to enter, long-press to select.

◫

Multi-select

Select any number of files and folders. The selection toolbar shows your count and batch actions inline.

⛓

Rule Chains

Compose multiple rename rules that apply in sequence. The output of each rule feeds into the next.

◉

Live Preview

See exactly what every file will become before committing. Conflict indicators flag collisions inside the batch or with existing files.

▶

Foreground Renaming

Renames run in a Foreground Service so operations survive app backgrounding. A live progress dialog shows per-file status.

↺

History & Undo

Every batch is journaled to a local Room database. Full-text search across old and new names. One-tap undo per batch.

⤓

Copy Mode

Create renamed copies and keep originals untouched, or rename in place. Your choice, every batch.

◊

Privacy First

No analytics. No tracking. No network code. Everything happens on your device — and stays there.

Rules

Six rules. Infinite combinations.

Stack them in any order to build the rename you want.

Add Prefix
Prepend text to the filename.
Add Suffix
Append text before or after the extension.
Set Base Name
Replace the filename, optionally preserving the extension.
Replace Text
Find and replace — plain text or full regex, case-sensitive or not.
Change Extension
Swap or strip the file extension.
Add Numbering
Sequential numbers with configurable start, step, and zero-padding.
